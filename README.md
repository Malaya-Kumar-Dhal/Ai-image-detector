🚨 Real or Fake X — AI Image Detector

With the rapid rise of generative AI, it’s becoming harder to tell whether an image is real or AI-generated. To explore this problem, I built Real or Fake X, a web-based tool designed to analyze images and help identify if they might be AI-created.

🌐 Try it here: https://real-or-fake-x.netlify.app

🔎 What it does:
• Upload an image and scan it instantly
• Detect patterns that may indicate AI-generated content
• Simple and fast user interface
• Accessible directly from the browser

🛠 Built With:
HTML • CSS • JavaScript • AI detection concepts • Netlify deployment

This project is part of my exploration into AI forensics, web development, and practical AI applications. As AI-generated media becomes more common, tools like this can help raise awareness and improve digital verification.
